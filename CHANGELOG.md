## [0.1.0] - 2026-01-20

### Added
- Initial release
- Zoho Books API client with async/await support
- OAuth authentication with token refresh
- Support for all Zoho regions (US, EU, IN, AU, JP)
- Models for contacts, invoices, expenses, payments, items, taxes, and accounts
- Rate limiting and error handling
- Multipart file upload support
